# Sistema_de_deteccion_y_validacion_de_billetes_Peruanos
Sistema de clasificación y autenticación de billetes peruanos basado en visión artificial. Utiliza una red neuronal EfficientNet para identificar la denominación y familia, y un Autoencoder U-Net con SSIM que detecta falsificaciones evaluando anomalías en la microtextura del papel.
